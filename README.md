MoonRaker-EM Rover side Application
===================================

C++ app for MoonRaker-EM.

## REQUIREMENT
32bit compiler which supports C++11

## BUILD
at the same directory of README.md

   cmake .
   make

and you will get two executables
   bin/TheSimple.exe
   bin/TimerRunner.exe
